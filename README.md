# SASS-Essentials
Beginners need to know about SASS

## 1. Store Data with SASS Variables
   It start with dollar __\(\$\)__ symbol followed by the variable name.
   
   __Examples__
   >\$main-fonts: Arial, sans-serif; 
   
   \/\/ To use variables:
   h1 \{
      font-family: \$main-fonts;
   \{
