## 1. Store Data with Sass Variables
   It start with dollar **\(\$\)** symbol followed by the variable name.
   
   **Examples**
   >\$main-fonts: Arial, sans-serif; 
   
   \/\/ To use variables:
   >h1 \{
      font-family: \$main-fonts;
   \}
